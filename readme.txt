Tweaks for GeneratePress. Menu - Multiple color options. Footer bar - disable, replace text, change colors. Header - change background, title, site description color.

== Description ==


Tweaks for GeneratePress, perhaps the best theme available.

HEADER - modify the following options:
Header background color.

TITLE / TAGLINE - modify the following options:
Title color.
Tagline color.

MENU - modify the following menu options:
Full Width Menu background color.
Constrained Width Menu background color.
Selected Menu Item.
Hover over Selected.
Hover over UNSelected.
Dropdown menu items color.
Dropdown menu items selected.
Dropdown menu items hover.
Menu font size.
Menu font color.

FOOTER - modify the following footer settings:
NOTE: The default copyright in footer is REMOVED by activation of the GP Tweaks plugin.
Hide / Unhide Footer via CSS.
Add text / HTML to the footer.
Change text color.
Change link and visited link text color.

BACK TO TOP BUTTON - modify the following options:
Change button color.
Change hover color.

Suggestions? Let me know.

== Installation ==

This section describes very briefly how to install the plugin and get it working.

e.g.

1. Upload the plugin file(s) to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. There is no setting menu.
4. Open Customizer, there is an option called GP Tweaks. Inside are options to make tweaks to the GeneratePress theme.
